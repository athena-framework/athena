Robust dependency injection service container framework.

## Installation

1. Add the dependency to your `shard.yml`:

```yaml
dependencies:
  athena-dependency_injection:
    github: athena-framework/dependency-injection
    version: ~> 0.3.0
```

2. Run `shards install`
